# Restaurant Website Template

## Description

A responsive restaurant website template developed using HTML and CSS. The project provides a clean and modern user interface suitable for restaurant businesses.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive design
* Attractive homepage layout
* Navigation bar
* Food menu section
* About section
* Contact section
* Modern UI design
